# Python Expense Tracker 📊

A simple, user-friendly Python application designed to help individuals log, manage, and analyze their daily financial expenses. This project was built as a mini-project to demonstrate data handling, file operations, and basic user interface design in Python.

## ✨ Features
* **Add Expenses:** Log amount, category (e.g., Food, Rent, Travel), date, and description.
* **View History:** Display a clean summary of all past transactions.
* **Category Filtering:** Filter and search expenses by specific categories or date ranges.
* **Monthly Budget Alerts:** Warns users when spending exceeds a predefined budget limit.
